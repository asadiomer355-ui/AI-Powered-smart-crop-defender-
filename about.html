Abstract

Recent advancements in machine learning (ML) and deep learning (DL) have revolutionized agricultural practices by enhancing plant disease detection — crucial for boosting agricultural productivity and ensuring food security. This project extends the current literature on ML and DL techniques, specifically focusing on their application via autonomous drones for real-time, image-based plant disease detection and targeted pesticide application.

We utilize convolutional neural networks (CNNs) for their proven effectiveness in identifying and classifying plant diseases from leaf images. The integration of Internet of Things (IoT) technology allows these drones to perform real-time data acquisition and precise actions, utilizing comprehensive datasets like PlantVillage to improve diagnostic capabilities significantly.

The architecture not only detects diseases using ML but also responds by applying pesticides using IoT technology. Comparative analyses reveal that deep learning architectures excel in handling complex image data, achieving high accuracy in disease identification. Future research will focus on enhancing model interpretability and scalability while refining drone navigation strategies to ensure efficient and environmentally friendly pesticide application.
